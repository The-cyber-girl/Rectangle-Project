# Rectangle-Project
A simple Java program demonstrating OOP concepts using a Rectangle class with constructors, methods, and object manipulation
This is a beginner-friendly Java project that demonstrates basic Object-Oriented Programming (OOP) concepts using a Rectangle class.

## Features
- `Rectangle` class with attributes for length and width.
- Constructor to initialize rectangle objects.
- Methods to calculate area and perimeter.
- Demonstrates creating objects, calling methods, and displaying results.

## How to run
1. Compile `Rectangle.java` and `Main.java`.
2. Run `Main.java` to see the output for rectangle dimensions, area, and perimeter.

## Example Output
- Shows length, width, area, and perimeter of rectangles created.
